# Sternmeister — AI Automation

EdTech-платформа в Германии, помогает мигрантам начать карьеру в бухгалтерии. Практическое обучение через симуляции реальных рабочих задач, AI-инструменты для подготовки резюме, сопроводительных писем и собеседований. Цель — обучить 15 000 мигрантов за 5 лет, 100 000 за 10 лет.

Фаундер — Александр Соловьёв ("Соло" из Кумар & Соло). Бэкграунд: Qmarketing, Qacademy (продан Skyeng), Refocus (EdTech в Юго-Восточной Азии). Sternmeister — новый EdTech-проект в Германии.

Этот репозиторий — рабочее пространство AI Automation Engineer.

## Миссия роли

Превратить AI и автоматизацию из модного концепта в практический инструмент, который экономит время и ресурсы бизнесу.

## Направления работы

### AI-ассистенты и чат-боты
- AI-ассистенты для поддержки студентов и отдела продаж
- Интеграция баз знаний с AI-поиском
- Системы ответов на частые вопросы

### Автоматизация процессов
- Настройка воркфлоу в n8n и аналогичных платформах
- API и webhook-интеграции между SaaS-сервисами
- Автоматизация коммуникаций: уведомления, напоминания, follow-up

### Аналитика и коммуникации
- Речевая аналитика для оценки качества звонков
- AI-driven outbound-системы и сценарии контакта
- Выявление паттернов в данных коммуникаций

### Эксперименты
- Тестирование новых AI-инструментов и оценка их применимости
- Быстрое прототипирование и проверка гипотез

## Ожидаемые результаты

- Снижение ручной нагрузки на команды
- Работающие AI-решения и автоматизации, которыми реально пользуются

- [Гайды](docs/4.%20guides/) — конвенции и процессы

### 2. Запуск через Docker (единственный поддерживаемый способ)

```bash
# 1. Настроить переменные окружения
cp .env.example .env
# Заполнить .env реальными значениями (KOMMO_TOKEN, WAZZUP_API_KEY и т.д.)

# 2. Собрать образ
docker build -t whatsapp-notifications .

# 3. Запустить контейнер
docker run -p 8000:8000 --env-file .env whatsapp-notifications

# 4. Проверить health check
curl http://localhost:8000/health
```

**Ожидаемый ответ:**
```json
{
  "status": "ok",
  "send_window": "9-21"
}
```

> **Локальный запуск без Docker не поддерживается.** Системный `python3` = 3.7, код требует 3.11+. Всегда используйте Docker.
