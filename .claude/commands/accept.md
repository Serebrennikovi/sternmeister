Пользователь акцептовал текущую задачу. Выполни ВСЕ шаги закрытия БЕЗ ВОПРОСОВ:

## Обязательные шаги

1. **Обновить статус задачи** → `done` в файле задачи
2. **Переместить файл задачи** с суффиксом `_done`:
   - Из `docs/3. tasks/S01_whatsapp_auto_notifications/` → `docs/3. tasks/Done/S01_whatsapp_auto_notifications_done/`
   - Добавить суффикс `_done` к имени файла
3. **Очистка папки спеки**:
   - Проверить `docs/3. tasks/S01_whatsapp_auto_notifications/` после перемещения
   - Если папка пустая → **удалить её** (rmdir)
4. **Обновить спецификацию** (S01 файл в `docs/2. specifications/`):
   - Ссылка на задачу → новый путь в `Done/` с `_done`
   - Статус задачи → ✅
   - История изменений → добавить запись
   - Если все задачи спеки выполнены → переименовать файл спеки с суффиксом `_done`
5. **Обновить HANDOFF.md** (`docs/HANDOFF.md`):
   - Обновить статус задачи на ✅ done
   - Добавить результат выполнения
   - **Если ВСЕ задачи спеки выполнены** → компактифицировать блок спеки
6. **Добавить запись в CHANGELOG.md** (`CHANGELOG.md`):
   - Секция `## [Unreleased]` → `### Added` (или создать если нет)
   - Формат: `- YYYY-MM-DD — T##: краткое описание что сделано`

## Структура проекта

```
docs/3. tasks/
├── S01_whatsapp_auto_notifications/  # Активные задачи по спеке S01
│   └── T##_s01_*.md
└── Done/                             # Завершённые задачи
    └── S01_whatsapp_auto_notifications_done/
        └── T##_s01_*_done.md
```

- Активные задачи: `docs/3. tasks/S01_whatsapp_auto_notifications/T##_s01_название.md`
- Завершённые: `docs/3. tasks/Done/S01_whatsapp_auto_notifications_done/T##_s01_название_done.md`
- Спецификации: `docs/2. specifications/S01_whatsapp_auto_notifications.md`

## Крайние случаи

| Ситуация | Действие |
|----------|----------|
| Задача без спеки (T00) | Перемещать в `Done/` напрямую, без подпапки спеки |
| Папки `Done/S01_*_done/` нет | Создать папку перед перемещением |
| Задача уже в Done | Пропустить, сообщить "уже акцептована" |
| Нет спецификации (S-файла) | Пропустить шаг 4 |
| Файл с таким именем уже есть | Предупредить, не перезаписывать |
